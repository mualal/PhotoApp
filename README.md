# PhotoApp

Данное приложение является тестовым. Для его использования достаточно запустить код без необходимости устанавливать дополнения.

В приложении представлены экраны входа и регистрации, выбора фотографии профиля пользователя из библиотеки Фото, экран со списком пользователей (сгенерированы случайным образом)
